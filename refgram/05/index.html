Toaq’s sentence structure can be described by the following diagram.

{% include gloss.html toaq="Keo súhu bı jỉa kỉaı jí hó nha." gloss="but the-pig TOP FUT taake.care I them PRM" translation="But as for the pigs, I promise to take care of them." audio="/letters/sounds/oaq.mp3" %}

Out of the indicated components, almost all are optional. The only obligatory components, which are present in every sentence, are the verbal complex and illocution (and, technically, the complementizer, which is usually covert in declarative and non-polar interrogative matrix clauses).

The minimal structure therefore looks like this:

{% include gloss.html toaq="Mả rủqshua moq?" gloss="C[+int] rain INT" translation="Is it raining?" %}

Each constituent will be dealt with in detail over the course of this description.
